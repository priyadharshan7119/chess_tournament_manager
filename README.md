# ♟️ Chess Tournament Management System

A Flutter application developed for the **Bytelogik Technical Assessment**.

## Features

* Player CRUD
* Tournament CRUD
* Add Players to Tournament
* Random Match Generation
* Tournament Rankings (Top 3)

## Tech Stack

* Flutter
* Dart
* Riverpod
* SQLite
